📖 Welcome Personnel

```

Welcome to the Phantom Realm Authority.

This handbook contains official policies, safety guidance, containment procedures, and behavioural expectations for all personnel.

Compliance is mandatory.

Reading is recommended.

Survival is encouraged.

```

1. Employee Conduct

```
  • Treat all personnel with professionalism.
  • Report unusual activity immediately.
  • Maintain accurate incident records.
  • Do not antagonise entities.
  • Do not challenge entities to physical combat.
  • Do not accept dares from entities.
  • Do not become emotionally attached to entities.
```

2. Containment Procedures

```
  • Follow authorised containment protocols.
  • Carry emergency documentation at all times.
  • Never enter a restricted area without approval.
  • If an anomaly speaks your name, leave immediately.
  • If an anomaly speaks someone else's name, leave immediately.
  • If an anomaly speaks its own name, run.
```

3. Research Division Guidance

```
  • All experiments require approval.
  • All experiments require documentation.
  • All experiments require a risk assessment.
  • Research notes should remain legible.
  • Frankenstein Duck may no longer approve his own experiments.
  • This policy exists due to prior incidents.
```

4. Duckpartment Regulations

```
  • The Infernal Duck is not authorised to create new departments.
  • The Infernal Duck is not authorised to create oversight committees.
  • The Infernal Duck is not authorised to create oversight committees for oversight committees.
```

5. Security Policy

```
  • Keep containment areas secured.
  • Report all breaches immediately.
  • Do not leave restricted areas unlocked.
  • Do not leave T-Rex Duck unsupervised near heavy machinery.
  • Do not ask why this rule exists.
```

6. Internal Affairs

```
  • All misconduct will be investigated.
  • All corruption reports will be reviewed.
  • All disciplinary actions will be documented.
  • Chuckie Duckie is not permitted to investigate incidents involving Chuckie Duckie.
  • This policy remains under review.
```

7. Executive Authority

```
  • The Supreme Director's decisions are final.
  • The Supreme Director is Moo Moo.
```

8. Workplace Safety

```
  • Never investigate an anomaly alone.
  • Never investigate an anomaly without documentation.
  • Never investigate an anomaly because "it seemed funny at the time".
  • If The Broadcast starts speaking directly to you, contact Containment Command.
  • If The Broadcast starts complimenting you, contact Containment Command faster.
```
